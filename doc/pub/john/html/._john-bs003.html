Efficient algorithms for bit manipulations

In addition to the existing material, there are some useful articles and references for the first steps.

The first step is to study efficient representations of the Slater determinants for words with more than 64 bits, allowing thereby for shell-model studies of systems with more than one major shell. Furthermore, the setup of the Hamiltonian matrix elements plays an important role.

In order to study these aspects and write an efficient program, the following articles can be of interest

  • Scemana's article on efficient implementation of the Slater-Condon rules
  • Simen Kvaal's article on developing an FCI code (open source code) for quantum dots
