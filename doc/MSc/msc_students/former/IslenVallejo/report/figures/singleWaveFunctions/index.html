Scattering of a Particle off of a Potential Barrier

  • MATLAB code. It takes about 5-10 minutes for the movies to be generated.
  • Movie. Note that this is a gzipped MATLAB MAT file. The expanded file is roughly 300MB in size. There are three movies saved in the file:
    • Incident wavefunction only (M1)
    • Reflected and transmitted wavefunction only (M2)
    • Total wavefunction (M3)
    To play the movies, use the MATLAB command movie(M), where M is the movie that you wish to play.
